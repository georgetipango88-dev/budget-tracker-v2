# Budget Tracker V2

## Project Overview

Budget Tracker V2 is a simple HTML and CSS web application designed to help users organize and visualize their daily expenses. The project demonstrates the use of semantic HTML, multimedia, forms, tables, and modern CSS styling techniques to create a clean, responsive, and user-friendly interface.

---

## Features

- Add Expense form
- Expense table with sample data
- Category dropdown menu
- Budget Tracker logo
- Embedded YouTube budgeting tips video
- Interactive "How to use this tracker" section
- Responsive and modern layout
- Google Fonts (Poppins & Roboto)
- Professional color palette
- Styled buttons, forms, and tables
- Hover effects and input focus styling
- CSS Box Model for spacing and layout
