# Budget Tracker V2

## Project Overview

Budget Tracker V2 is a simple HTML and CSS web application designed to help users organize and visualize their daily expenses. The project demonstrates the use of semantic HTML, tables, forms, multimedia, and modern CSS styling techniques.

---

## Features

- Add Expense form
- Expense table with sample data
- Category dropdown menu
- Budget Tracker logo
- Embedded YouTube budgeting tips video
- Interactive "How to use this tracker" section
- Responsive and clean layout
- Modern CSS styling
- Hover effects
- Advanced CSS selectors
